# Cinepuma

## ¿Qué es Cinepuma?
Cinepuma es una aplicación web donde puedes opinar sobre cientos de películas al igual que ver las opiniones de los demás. Al registrarte, puedes poner comentarios y dar nota a las películas que más te interesen. Las puedes buscar tanto con un buscador como por categorías.

## ¿Cómo abro el proyecto?
Mediante la extensión de Visual Studio Code llamada Live Server puedes acceder sin problema a la aplicación
